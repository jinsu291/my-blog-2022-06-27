1. Math

- Math는 생성자 함수가 아니다. Math는 정적 프로퍼티와 정적 메서드만 제공한다.

- Math.PI = 3.141592653589793

- Math.abs (인수로 전달된 숫자의 절대값을 반환)

- Math.round (인수로 전달된 숫자의 소수점 이하를 반올림한 정수)

- Math.ceil (인수로 전달된 숫자의 소수점 이하를 올림한 정수)

- Math.floor (인수로 전달된 숫자의 소수점 이하를 내림한 정수)

- Math.sqrt (인수로 전달된 숫자의 제곱근을 반환)

- Math.random (임의의 난수 반환 0<= <1)

- Math.pow (첫번째 인수를 밑, 두 번째 인수를 지수로 거듭제곱한 결과)

- Math.min (전달받은 인수 중에서 가장 작은 수 반환)

- Math.ceil (인수로 전달된 숫자의 소수점 이하를 올림한 정수)

2. Date

- Date 생성자 함수를 인수없이 new 연산자와 함께 호출하면 현재 날짜와 시간을 가지는 Date객체를 반환한다.

- new 연산자 없이 호출하면 Date 객체를 반환하지 않고 날짜와 시간 정보를 나타내는 문자열을 반환한다.

- new Date(miliseconds) (스타트점(1970년 1월 1일 0시0분0추) 기준 경과한 초를 날짜와 시간을 나타내는 Date 객체 반환)

- new Date(dateString) (인수로 전달한 문자열은 Date.parse 메서드에 의해 해석 가능한 형식이어야 함)

- new Date(숫자) (월은 실제로 하나를 올려야한다.)

- Date.now (스타트점 기준 밀리초 반환)

- Date.parse(스타트점 기준 지정시간 차이 밀리초 반환)

- Date.prototype.get**\_ \_** -> FullYear, Month, Date, Day, Hours, Minutes, Seconds, Milliseconds, Time -> 반환 (이때 월은 1줄어서 반환, 일요일 0부터 시작 주의!!)

- Date.prototype.get**\_ \_** -> FullYear, Month, Date, Day, Hours, Minutes, Seconds, Milliseconds, Time -> 설정 (이때 월은 1줄어서 세팅 주의!!)

- Time 은 밀리초로 반환

- 나머지는 인터넷 검색할 것
