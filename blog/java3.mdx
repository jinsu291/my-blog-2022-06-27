//자바
class OperatorEx1 {
public static void main(String args []) {
int i = 5, j = 5;
System.out.println(i++); //5
System.out.println(++j); //6
System.out.println(“i = ” + i + “, j = ” + j); // i = 6, j = 6
}
}

- 전위형은 변수의 값을 먼저 증가시킨 후에 변수의 값을 읽어오는 반면, 후위형은 변수의 값을 먼저 읽어온 후에 값을 증가시킨다.

//자바
char c1 = ‘a’;
char c3 = ‘ ’;
c3 = (char)(c1 + 1);  
c1++;

- 덧셈연산 c1+1의 결과가 int이므로 이 결과를 char형 변수 c3에 담기위해서는 char형으로 형변환이 필요

- c1++대신에 c1 = c1 + 1;을 사용하면 에러가 발생한다. c1 + 1의 연산결과는 int이며, 그 결과를 다시 c1에 담으려면 형변환 연산자를 사용하여 char형으로 형변환해야하기 때문
