- 숫자 타입이 아닌 피연산자에 + 단항 연산자를 사용하면 피연산자를 숫자타입으로 변환하여 반환한다.

//자스
var x = ‘1’;
console.log(+x); //1
console.log(x); //“1”
x = true;
console.log(+x); //1
console.log(x); //true
x = ‘Hello’;
console.log(+x); //Nan
console.log(x); //“Hello”

- 개발자의 의도와는 상관없이 자바스크립트 엔진에 의해 암묵적으로 타입이 자동변환되기도 한다.

- 동등비교(==)연산자는 좌항과 우항의 피연산자를 비교할 때 먼저 암묵적 타입 변환을 통해 타입을 일치시킨 후 같은 값인지 비교한다. 때론 편리한 경우도 있지만 결과를 예측하기 어렵고 이해하지 않으려 해도 된다. 대체로 사용하지 않는 것이 좋다.

- 일치비교(===)연산자는 좌항과 우항의 피연산자가 타입도 같고 값도 같은 경우에 한해 true를 반환한다.

- 자바스크립트에는 양의 0과 음의 0이 있는데 이들을 일치비교하면 true를 반환한다.

//자스
!0; //true
!‘Hello’; //false

- 논리부정 연산자는 언제나 불리언 값을 반환한다. 만약 피연산자가 불리언 값이 아니면 불리언 타입으로 암묵적 타입 변환한다.

//자스
2 \*\* 3; //8

- 지수연산자는 좌항의 피연산자를 밑, 우항의 피연산자를 지수로 거듭 제곱하여 숫자 값을 반환
